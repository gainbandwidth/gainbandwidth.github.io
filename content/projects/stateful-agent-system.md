## 项目背景

随着大语言模型能力的提升，AI Agent（智能体）成为将 LLM 能力落地到复杂任务场景的关键范式。本项目设计了一个基于**状态维护**的智能体系统，能够自主完成多步骤、多工具的复杂任务。

## 系统设计

### 多步任务执行流水线

采用 LangGraph 构建有向图式的任务执行流程，每个节点代表一个工具调用或决策步骤，支持条件分支和循环重试。

### 工具集成

- **Bash 执行**：在沙箱环境中执行 Shell 命令
- **网页抓取**：自动获取和解析网页内容
- **MCP 工具链**：通过 Model Context Protocol 接入外部服务

### 记忆模块

设计了长期记忆（Long-term Memory）机制，使智能体能够跨任务会话保持上下文连续性，避免重复推理。

### 动态任务调度

基于任务优先级和资源状态动态调度执行，支持并行执行独立子任务。

## 核心成果

- 实现了端到端的多工具协同任务执行
- 长期记忆模块显著减少重复推理开销
- 系统具备良好的可扩展性，可灵活接入新工具

## 技术栈

`Python` `LangChain` `LangGraph` `MCP` `Agent Tooling`
