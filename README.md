# 顾博文的技术博客

基于 [Hugo](https://gohugo.io/) + [PaperMod](https://github.com/adityatelange/hugo-PaperMod) 主题搭建的个人博客。

访问地址：[https://gainbandwidth.github.io](https://gainbandwidth.github.io)

## 内容板块

- **博客** — 学习笔记与技术文章
- **项目** — 个人项目展示
- **随笔** — 个人思考与生活记录
- **资源** — 常用工具与学习资源推荐

## 本地开发

```bash
# 安装 Hugo
brew install hugo

# 本地预览
hugo server -D
```

## 部署

通过 GitHub Actions 自动构建部署到 GitHub Pages，推送代码到 `main` 分支即可自动更新。
